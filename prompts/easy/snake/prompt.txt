请使用 Python 3 和 pygame 实现一个可直接运行的 Snake Easy 游戏，并严格遵守以下固定规范，以保证不同模型输出的稳定性和可比性。

通用约束：
1. 只输出完整 Python 代码，不要输出解释、Markdown、代码块标记。
2. 代码必须单文件可运行，不能依赖任何本地图片、音频、字体或其他外部资源。
3. 使用 `pygame`，窗口大小固定为 `800x600`，帧率固定为 `60 FPS`。
4. 所有可调参数都写成顶部常量，随机行为使用 `random.seed(42)`。
5. 必须支持关闭窗口退出、`R` 重新开始、`ESC` 退出。
6. 必须显示标题、当前分数和游戏结束提示。

玩法与布局固定要求：
1. 游戏主区域使用固定网格：`30 列 x 24 行`，每格 `20x20` 像素，网格区域大小为 `600x480`，居中显示。
2. 蛇初始长度固定为 `3`，初始位置在网格中心附近，初始方向向右。
3. 玩家使用方向键控制蛇移动，不允许直接反向掉头。
4. 蛇的移动速度固定为每秒 `10` 个网格。
5. 食物始终只存在 `1` 个，大小为 `1` 个网格，必须生成在蛇身体之外。
6. 每吃到 `1` 个食物，蛇长度 `+1`，分数 `+10`。
7. 蛇撞到墙壁或撞到自己时立即游戏结束，不允许穿墙。
8. 游戏结束后画面停止在最终状态，并显示 `Game Over`、最终分数和 `Press R to Restart`。

视觉要求：
1. 背景、网格、蛇头、蛇身、食物颜色要明显区分。
2. HUD 放在网格区域上方或左上角，不遮挡主体玩法区域。
3. 不要加入额外玩法，例如道具、加速、传送、关卡、音效。
